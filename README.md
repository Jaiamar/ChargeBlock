# EVConnect TN (ChargeBlock)

A dynamic and highly interactive EV Charging Locator and Booking Prototype for Tamil Nadu.

## Features
- **Live Interactive Map:** Visualizes EV stations across Tamil Nadu using Leaflet.
- **Real-Time Telemetry:** Displays current status of chargers (Free, Used, Offline).
- **Advance Booking:** Allows scheduling and blocking a slot using simulated pre-authorization.
- **Premium Aesthetics:** Dark mode glassmorphism UI for a cutting-edge aesthetic.

## Setup
```bash
npm install
npm run dev
```
